# ESL HW1 Part1

## Gaussian blur filter with FIFO channels

### How to execute the code?

1. Compile the program
   '''shell
   $ cd FIFO
   $ mkdir build
   $ cd build
   $ cmake ..
   $ make
   '''
2. Run the program
   '''shell
   make run
   '''
3. Resulting image is saved as './build/out.bmp'
